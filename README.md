Dream Entropy Project - Full Package

Contents:
- simulation.py
- run_simulations.py
- requirements.txt
- ISEF_report_outline.md

How to run:
1. (Optional) Create virtual env:
   python -m venv venv
   source venv/bin/activate   # or venv\\Scripts\\activate on Windows

2. Install:
   pip install -r requirements.txt

3. Run:
   python run_simulations.py

Outputs:
- dream_simulation_summary.csv
- figures/entropy_vs_time.png
- figures/phase_rem.png
- figures/phase_lucid.png
- figures/heatmap_rem.png
- figures/heatmap_lucid.png
